Los formularios se incluyen dentro de la etiqueta forms

En ellos se utilizan los siguientes controles

  • Imput - text
  • text area
  • checkbox
  • radio buttons
  • select /options
  • botones

La etiqueta form tiene dos atributos importantes

  • action
  • method
